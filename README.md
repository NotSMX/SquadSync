# SquadSync

## Gmail (sending emails)

Set in your environment (e.g. `.env`):

- `EMAIL_USER` – your full Gmail address (e.g. `you@gmail.com`)
- `EMAIL_PASSWORD` – a **Gmail App Password**, not your normal password

To create an App Password: Google Account → Security → 2-Step Verification (turn on if needed) → App passwords → generate one for “Mail”. Use that 16-character password as `EMAIL_PASSWORD`.
